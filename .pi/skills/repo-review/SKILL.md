# Repo Review

Use this skill when the task is a code review or change audit.

Workflow:

1. Read `.pi/prompts/review.md`.
2. Inspect the diff or the touched files.
3. Prioritize findings by severity.
4. Report defects, regressions, and missing validation before summaries.

Outputs:

- Findings with file references.
- Open questions when behavior is ambiguous.
- Brief residual-risk summary if needed.
